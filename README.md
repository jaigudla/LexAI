# LexAI

LexAI is a full-stack, cloud-native legal document analysis platform that allows users to securely upload legal contracts and automatically extract structured insights using AI.

## Project Structure

- `frontend/`: React + TypeScript application
- `backend/`: Spring Boot application
- `infrastructure/`: AWS deployment configurations

## Key Features

- Secure Document Upload
- AI-Powered Insights (OpenAI)
- Asynchronous Processing
- Scalable Architecture

## Getting Started

### Backend
Navigate to `backend/` and run:
```bash
./mvnw spring-boot:run
```

### Frontend
Navigate to `frontend/` and run:
```bash
npm install
npm run dev
```
